#### 一、使用方法

##### 1、需要复制的内容是固定不变的

```
<li class="weixin" v-clipboard:copy="copyMessage" v-clipboard:success="successCopy" v-clipboard:error="errorCopy">
  <span>微信联系</span>
</li>
```

```
v-clipboard:copy --->  需要复制的信息
successCopy      ---> 成功回调
errorCopy        ---> 失败回调
```

##### 2、需要复制的内容是动态变化的

```
<p>{{copyMessage}}</p>
<button type="button" @click="copy('copy content')">Copy!</button>
```
